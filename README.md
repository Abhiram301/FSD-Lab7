# 🔐 FSD Lab 7 — AR/VR with A-Frame

A web-based **Augmented Reality (AR) and Virtual Reality (VR)** experience built with A-Frame and React + Vite.

## ✨ Features
- Interactive 3D AR scene using A-Frame
- 360° VR classroom environment
- PWA support for mobile installation
- Fetches live data from a public API

## 🛠️ Tech Stack
- React 19
- Vite
- A-Frame
- Vanilla CSS

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
